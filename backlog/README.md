# Backlog - Mega Proyectos

Esta carpeta contiene los "mapas de elefantes" - proyectos grandes que se ejecutan en múltiples semanas.

## Estructura
Cada archivo `.json` representa un mega-proyecto activo.

## Reglas
- Solo proyectos que tomen más de 3 semanas van aquí.
- El Coach lee estos archivos para saber qué "hormiga" sugerir cada semana.
- Tú NO editas estos archivos manualmente. El Coach los actualiza.
